Hobi Saya


Hobi saya adalah bermain game, game pertama saya adalah Point Blank, mulai dari situ saya mengenal dan mempunyai banyak teman dari Sabang sampai Merauke dan dari pulau Miangas hingga ke pulau Rote.


Saya sangat senang bermain game FPS/perang-perangan karena saya dulu bercita-cita menjadi Tentara, Pada Tanggal 1 Desember 2018 game Point Blank Gemscool tutup dan pindah ke Garena.


Mulai dari situ saya mengenal game kedua saya yaitu Minecraft, saya sangat menyukai Game Minecraft karena membuat saya bercita-cita menjadi Arsitek bukan Iron-Man.


Saya sangat Ahli membuat bangunan di Minecraft, saya sudah membuat berbagai macam bangunan mulai dari Rumah saya sampai Rumah Tetangga yang berada dibelakang Rumah saya.


Pada tahun 2019 saya diajak mantan Teman saya yang dahulu seangkatan bermain Point-Blank untuk bergabung ke Servernya dan mulai saat itu saya mengenal apa itu Server dan saya ditugaskan untuk menjadi Developer di servernya.


Lalu membuat saya menjadi tertarik kepada Dunia IT karna saya berprinsip "Kalau ada yang susah kenapa harus mudah?", lalu saya belajar dengan Otomotif eh Otodidak melalui platfrom video online seperti Youtube.


Saya dengan giat belajar cara membuat Programing hingga larut malam, Saya mulai mengetahui sedikit-dikit cara membuat HTML,PHP,CSS,Script dan Java dari Video-Video yang ada di Youtube walau saya tidak terlalu mengerti Bahasa Inggris.


Dan pada Tahun 2020 saya diterima di sekolah SMAN 1 CIBARUSAH dan memilih untuk bergabung menjadi anggota Eskul IT untuk mengembangkan Ilmu Programing saya.


Sekian dari Saya, Mohon maaf bila banyak kekurangan dan Mohon maaf bila tidak lucu karena saya sedang tidak melawak tetapi sedang menceritakan tentang Hobi saya hingga mengenal Dunia IT.
